# Phase 5: 图表可视化

## 需求

Agent 返回查询结果时，如果包含数值列，用户可切换查看图表。

## 实现方案

### 方案：纯前端解析（不改造后端）

Agent 返回的 Markdown 表格，前端 Chat 组件自行解析，检测到数值列时展示图表按钮。

```
Agent 返回:
"| 产品名 | 销售额 |
  |--------|--------|
  | 山地车 | 120000 |
  | 公路车 | 98000  |"

                 ↓
前端解析 → 检测到数值列 "销售额"
         → 表格上方出现 "📊 图表" 切换按钮
         → 点击后渲染柱状图
```

### 优点
- 后端不动，Agent 逻辑不改
- Agent 返回什么都能尝试解析
- 图表是后加的，不影响已有功能

### 图表库选型
| 库 | 体积 | 适合 |
|----|------|------|
| Recharts | 小 (200KB) | React 友好，API 简单 |
| ECharts | 大 (1MB) | 图表类型多，但太重 |
| Chart.js | 中 (300KB) | 轻量，够用 |

推荐 **Recharts** — React 原生，柱状图/折线图/饼图够用，不拖慢首屏。

### 前端改动点

| 文件 | 改动 |
|------|------|
| `Chat.tsx` | 消息渲染时检测 Markdown 表格 + 数值列 |
| `ChartViewer.tsx` (新) | 图表渲染组件，支持柱状图/折线图/饼图切换 |
| `package.json` | 加 recharts 依赖 |

### 自动图表类型推断

| 数据结构 | 自动推荐的图表 |
|---------|-------------|
| 1 个分类 + 1 个数值 | 柱状图（默认） |
| 1 个时间维度 + 1 个数值 | 折线图 |
| 1 个分类 + 多个数值 | 分组柱状图 |
| 1 个分类 + 占比数值 | 饼图选项 |

### 交互

- 表格下方/上方显示图表按钮
- 点击后原地展开图表（不跳转）
- 图表可切换类型（柱状/折线/饼图）
- 导出图片功能可选
